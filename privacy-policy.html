# Datenschutzerklärung (Deutsch) Stand: Juni 2026 ## 1. Verantwortlicher Verantwortlich für die Datenverarbeitung im Zusammenhang mit dem Discord-Bot „Larsium V2“ ist: Betreiber: knirpslarsi_yt Kontakt: [support@knirpslarsi.de](mailto:support@knirpslarsi.de) ## 2. Erhobene Daten Larsium V2 verarbeitet ausschließlich Daten, die für die Bereitstellung seiner Funktionen erforderlich sind. Dazu können gehören: * Discord User IDs * Discord Server IDs * Discord Channel IDs * Discord Rollen-IDs * Servereinstellungen und Bot-Konfigurationen ## 3. Ticketsystem Bei Nutzung des Ticketsystems können Ticket-Inhalte innerhalb des jeweiligen Discord-Servers gespeichert werden. Dies dient ausschließlich der Bearbeitung und Dokumentation von Supportanfragen. ## 4. Geburtstagsfunktion Nutzer können freiwillig ihr Geburtsdatum hinterlegen, um Geburtstagsbenachrichtigungen zu erhalten. Hierfür werden gespeichert: * Discord User ID * Geburtsdatum Die Nutzung dieser Funktion ist freiwillig und kann jederzeit beendet werden. ## 5. Autogrammkarten Nutzer können freiwillig ihre Postanschrift angeben, um eine Autogrammkarte zu erhalten. Die Anschrift wird ausschließlich an den Betreiber des Bots (knirpslarsi_yt) übermittelt und nur zum Versand der angeforderten Autogrammkarte verwendet. Nach Bestätigung des Erhalts wird die Anschrift gelöscht. Die Daten werden nicht öffentlich angezeigt und sind nicht über Konsolen-Ausgaben oder andere öffentliche Systeme einsehbar. ## 6. Weitergabe von Daten Personenbezogene Daten werden nicht verkauft oder an unbeteiligte Dritte weitergegeben. ## 7. Löschung von Daten Nutzer haben jederzeit das Recht, die Löschung ihrer gespeicherten Daten zu verlangen. Eine Anfrage kann per E-Mail an [support@knirpslarsi.de](mailto:support@knirpslarsi.de) oder über das Support-System von Larsium V2 gestellt werden. Nach Eingang einer berechtigten Anfrage werden die betreffenden Daten innerhalb einer angemessenen Frist gelöscht, sofern keine gesetzlichen oder technischen Gründe einer sofortigen Löschung entgegenstehen. Folgende Daten können auf Anfrage gelöscht werden: * Gespeicherte Discord User IDs * Geburtstagsdaten * Ticketbezogene Daten, soweit technisch möglich * Sonstige durch Larsium V2 gespeicherte nutzerbezogene Informationen Adressdaten, die für den Versand von Autogrammkarten übermittelt wurden, werden unabhängig von einer Löschanfrage nach Bestätigung des Erhalts der Autogrammkarte gelöscht. ## 8.
